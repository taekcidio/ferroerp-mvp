# FerroERP MVP — ERP para Ferreterías

**FerroERP** es un ERP liviano y modular (MVP) diseñado para ferreterías pequeñas a medianas (1–3 sedes, 2–10 empleados).  
Busca reemplazar Excel y procesos manuales para reducir pérdidas de inventario, errores en ventas y mejorar el control administrativo.

## Alcance del MVP (Módulos)
1. **Proveedores**: registro de proveedores, asociación de productos, compras e historial
2. **Inventario**: productos, control de stock, stock mínimo, kardex/movimientos, alertas por bajo inventario
3. **Ventas y Facturación**: venta rápida (mostrador), facturas, ventas de contado y crédito, actualización automática del inventario
4. **CRM (Clientes)**: registro de clientes, historial de compras, clientes frecuentes, control básico de crédito
5. **Usuarios y Roles (RBAC)**: creación de usuarios, roles (Admin/Vendedor/Bodeguero) y permisos
