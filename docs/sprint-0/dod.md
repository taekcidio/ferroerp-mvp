# FerroERP — Definition of Done (DoD) (Sprint 0)

Una historia está **Done** (terminada) si cumple:

- [ ] Cumple todos los **criterios de aceptación**
- [ ] Código subido al repositorio (commit y/o PR)
- [ ] **Pruebas ejecutadas**: mínimo 5 casos por historia
- [ ] Evidencia de pruebas adjunta (capturas o links)
- [ ] Documentación actualizada (README/docs)
- [ ] Listo para demo en Sprint Review (flujo feliz funcional)
