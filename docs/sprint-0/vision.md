# FerroERP — Visión del Proyecto (Sprint 0)

## Contexto del negocio
El proyecto se enfoca en la digitalización y optimización de una ferretería pequeña a mediana (una o varias sedes) que actualmente opera con procesos manuales o herramientas básicas como Excel.

La ferretería:
- Compra productos a múltiples proveedores.
- Maneja inventario físico en bodega.
- Vende productos al detal y al por mayor.
- Genera facturas por cada venta.
- Tiene clientes frecuentes.
- Cuenta con empleados (vendedores, bodegueros y administrador).

Problemas actuales:
- Errores en inventario y pérdidas por descontrol de stock.
- Duplicidad de información.
- Dificultad para generar reportes.
- Baja visibilidad de ventas y rentabilidad.

## Propósito del proyecto
Diseñar e implementar un ERP especializado para ferreterías que centralice y automatice procesos clave del negocio, garantizando control, trazabilidad y eficiencia operativa.

Características del sistema:
- Modular y escalable.
- Enfocado en PYMES.
- Simple y fácil de usar.
- Preparado para crecimiento futuro (multi-sede, facturación electrónica, BI).

## Alcance funcional (alto nivel)
El ERP cubrirá procesos principales:
- Gestión de Proveedores
- Gestión de Inventario
- Ventas y Facturación
- Gestión de Clientes (CRM básico)
- Gestión de Usuarios y Roles

Flujo a soportar:
Proveedor → Compra → Inventario → Venta → Facturación → Cliente

## Impacto esperado
Con la implementación del ERP, la ferretería podrá:
- Tener inventario en tiempo real.
- Reducir errores manuales.
- Mejorar la atención al cliente.
- Tomar decisiones basadas en datos.
- Incrementar rentabilidad.

## Sprint Goal (Sprint 1)
Entregar un incremento funcional que permita:
- Registrar productos (Inventario)
- Registrar entrada de stock
- Registrar salida de stock
- Consultar stock
- Registrar clientes (CRM)
- Consultar clientes
- (Opcional) Registrar interacción con cliente

## Fuera de alcance del Sprint 1
- Facturación avanzada y electrónica.
- Compras completas con proveedores (orden de compra + recepción completa).
- Nómina / RRHH.
- Reportería/BI avanzada.
